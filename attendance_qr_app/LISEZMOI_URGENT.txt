================================================================
                                                               
     SOLUTION RAPIDE - ERREUR INDEX FIREBASE            
                                                               
================================================================

----------------------------------------------------------------
                                                              
  ERREUR ACTUELLE :                                        
  [cloud_firestore/failed-precondition]                      
  The query requires an index                                
                                                              
----------------------------------------------------------------

----------------------------------------------------------------
                                                              
  CAUSE :                                                  
  Les index Firestore ne sont PAS deployes sur Firebase      
                                                              
----------------------------------------------------------------

----------------------------------------------------------------
                                                              
  SOLUTION EN 3 COMMANDES :                                
                                                              
  1. npm install -g firebase-tools                          
                                                              
  2. firebase login                                         
                                                              
  3. firebase deploy --only firestore:indexes               
                                                              
----------------------------------------------------------------

----------------------------------------------------------------
                                                              
  ALTERNATIVE (SI VOUS N'AVEZ PAS NPM) :                   
                                                              
  Cliquez sur le lien dans votre erreur :                    
  https://console.firebase.google.com/v1/r/project/...       
                                                              
  Cela creera automatiquement l'index manquant !          
                                                              
----------------------------------------------------------------

----------------------------------------------------------------
                                                              
  ENSUITE, LANCEZ L'APPLICATION :                          
                                                              
  flutter run
